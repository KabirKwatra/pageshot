# shot.now.sh

Generate screenshot from webpage.

## Usage

Navigate to [https://shot.now.sh/?url=https://google.com]

## Query Params

- __url__: The webpage location. (required)
- __width__: Viewport width. (default: 960)
- __height__: Viewport height. (default: 640)
